Wales' Most Famous Landmarks

Home Wales' Famous Landmarks Lighthouse Trail In Wales References

Using the Leaflet API I have displayed a map that shows three of Wales' most famous landmarks. Featuring the Principality Stadium, Snowdonia National Park and Pembroke Castle. Each landmark is shown on the map with a custom Welsh dragon flag. If the user clicks on one of these markers, a popup window is brought up displaying the name of the chose landmark.

Using Thunderforest web map services I have implemented third party tile servers, including a cycle map, a transport map and a dark mode transport map. Each of these raster layers is available for the user to select via an interactive menu in the top right hand corner.
